# Online SaLes dashboard


## Executive Summary:

  This project was taken from kaggle of a data set downloaded using html text. The text was converted to exel for data cleaning, properly deleting or giving nulls/missing values, getting rid of dulplications and finally combining the more bags value with the bags value. Then the data was transferred to tableua where a dashboard was created to show sales throughout the year, product type revenue each month and how much each product made throughout the year with an interactive filter on the graphs.


### Business Problem: 

  An online shopping company wants a dashboard that shows their profit each month and which product category makes the most money at the end and thrpughput the year.



### Methodology: 

1. Excel - data cleaning, transformation
   
2. Tableau - dashboard creation


### Skills:


Excel: data analysis functions, simple Excel cleaning functions (remove duplicates, filter, AVG)

Data visualization: dashboard, bar chart




### Results & Business Recommendation: 

  A succesful interactive board was created for stakeholders to view giving them sales throughout the year and how well product categories did during the year to help make better data-driven decisions.



What was implemented to complete the project:

1. Data was loaded and then transformed to an excel sheet
2. excel sheet was cleaned and transformed to tableau
3. Dashboard with the required metrics was created with an interactive filter for the online stores stakeholders.
